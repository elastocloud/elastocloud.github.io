Elasto

Warning: This project is no longer actively maintained and should not be used.

Elasto is a lightweight library and client utility for managing and manipulating cloud storage objects via REST protocols, offering:

  • Microsoft Azure account, container and blob modifications
  • Azure file share provisioning and access
  • Amazon S3 bucket and object manipulation
  • File/object upload, download and remote copy operations
  • A flexible, vaguely-POSIX-like API
